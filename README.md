## 🤖 Engenharia de Prompts e "Cicatrizes"
Durante os testes no NotebookLM, foram elaborados prompts estratégicos para extrair explicações claras e aplicáveis:

- **Prompt inicial:** "Explique os conceitos fundamentais para quem está começando em Python."  
  - **Resposta obtida:** Sintaxe, variáveis, operadores, estruturas de controle, funções, tratamento de exceções e orientação a objetos.  
  - **Dificuldade:** Resposta extensa, precisei organizar em tópicos.  

- **Prompt de variação:** "Explique docstrings em Python com exemplos práticos."  
  - **Resposta obtida:** Uso de aspas triplas, integração com `help()`, boas práticas de documentação.  
  - **Cicatriz:** Precisei ajustar o prompt para diferenciar docstrings de comentários comuns.  

- **Prompt de revisão:** "Qual é o resultado de 5 // 2?"  
  - **Resposta obtida:** 2, com explicação sobre divisão inteira e relação com operador módulo `%`.  

Esse registro mostra o raciocínio e o troubleshooting feito para chegar em respostas mais úteis.

## 📒 Miniguia de Estudo

### Resumos Estruturados
- **Docstrings:** Documentam funções, classes e módulos usando aspas triplas. Integradas ao sistema `help()`.  
- **Divisão Inteira (`//`):** Retorna apenas a parte inteira da divisão, útil em algoritmos e índices.  
- **Método `.split()`:** Divide strings em listas com base em separadores, essencial para limpeza de dados.  
- **Conceitos Fundamentais:** Sintaxe, variáveis, operadores, estruturas de dados (listas, tuplas, dicionários, sets), funções e orientação a objetos.

### Glossário
- **Indentação:** Espaços usados para estruturar blocos de código em Python.  
- **Variável:** Local na memória para armazenar dados.  
- **Módulo:** Arquivo com código reutilizável, importado com `import`.  
- **Overfitting:** Quando um modelo aprende demais os detalhes do treino e perde generalização.  

### Conjunto de Prompts Reutilizáveis
- "Explique [conceito] em linguagem simples."  
- "Mostre exemplos práticos de [função ou operador]."  
- "Crie um resumo em tópicos sobre [tema]."  
- "Monte um quiz com 5 perguntas sobre [assunto]."
